# 风格预设库（视频提示词 / 分镜表 适配）

> 借鉴 `storyboard-converter` 的风格预设（景别分布 / 运镜偏好 / 色调 / BGM 曲线）。
> 用于普通模式与适配模式共用，按题材自动匹配。

## 一、风格预设总表

| 预设 | 景别分布 | 运镜偏好 | 色调 | 适用 |
|------|---------|---------|------|------|
| 电影写实（默认） | 全景20%中景40%近景25%特写15% | 固定→推→移/跟 | 自然光，冷暖随场 | 严肃/史诗/悬疑 |
| 日系动漫（继承 style-log） | 中景35%近景30%特写20%全景15% | 固定→推→急推/急拉 | 高饱和，渐变天空 | 轻小说/热血/治愈 |

## 二、BGM 情绪曲线模板（分镜表可选列）

```
开场：轻柔渐入
推进：渐强 / 低沉微压（冲突段）
高潮：明快 / 强节奏
收束：渐弱回到平静
```

## 三、普通模式默认选择
- 未指定风格 → 默认「电影写实」；若检测到网文爽点密度高 → 提示可切「短视频推文」
- 继承 novel-master `style-log.md` 的「日漫风格增强」标记 → 自动启用「日系动漫」预设叠加独白镜/慢镜头

## 四、合规红线（借鉴 ark-novel-to-tuiwen 抖音合规）
| 红线 | 替代做法 |
|------|---------|
| 血腥暴力（喷血/断肢/伤口特写） | 用情绪结果代替：「他倒下了」不写血溅 |
| 色情擦边 | 感情戏停留牵手/对视/微笑 |
| 诱导关注（点关注/点赞） | 用悬念结尾代替 |
| 封建迷信（现实世界） | 虚构世界观内允许，不宣扬现实迷信 |
| 恐怖灵异（强烈不适） | 悬疑保紧张不出现恐怖形象 |
| 负面价值观（炫富/霸凌美化） | 逆袭"有钱"是打脸工具不写炫富 |
